# MARIE

An unofficial (incomplete) implementation of the MARIE machine architecture and
assembly language from the book _The Essentials of Computer Organization and
Architecture_ (2003) by Linda Null and Julia Lobur.

## Building

You need a C11 compiler to build this project. Get the source code of this
repository and run the following:

    cc -std=c11 -O2 -o marie main.c
